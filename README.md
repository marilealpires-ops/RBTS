# Rede Brasileira de Telessaúde

Projeto pronto para GitHub Pages.

## Como publicar

1. Extraia este ZIP.
2. Envie o arquivo `index.html` para a raiz do repositório `rbt`.
3. No GitHub, vá em `Settings > Pages`.
4. Escolha `Deploy from a branch`, branch `main`, pasta `/root`.
5. Salve e aguarde alguns minutos.

O site abrirá em:

`https://SEU_USUARIO.github.io/rbt/`

## Importante

Este projeto é um arquivo único (`index.html`) com dados, estilos e scripts embutidos. Isso evita erro de pastas quebradas no GitHub Pages.
